# Yashashvi Tiwari — Portfolio

Personal developer portfolio built with React and Vite.

🔗 **Live site:** [yashashvi25.github.io/yashashvi-portfolio](https://yashashvi25.github.io/yashashvi-portfolio/)

---

## Tech Stack

- React 18 + Vite 5
- CSS Modules
- Web3Forms (contact form)
- GitHub Pages (deployment)

---

## Run Locally

```bash
npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file in the project root:
VITE_WEB3FORMS_KEY=your_key_here
